CSS CHEAT SHEET

1. - Border

La propiedad de CSS de borde establece el borde de un elemento. Establece los valores de border-width, border-style y border-color.

2. - Border-top, Border-bottom, Border-right, Border-left

La propiedad de CSS border-left establece todas las propiedades del borde izquierdo de un elemento. La propiedad de CSS border-right establece todas las propiedades del borde derecho de un elemento. La propiedad de CSS border-top establece todas las propiedades del borde superior de un elemento. La propiedad de CSS border-bottom establece todas las propiedades del borde inferior de un elemento.

3. - Border-radius

La propiedad de CSS border-radius redondea las esquinas del borde exterior de un elemento. Puede establecer un solo radio para hacer esquinas circulares o dos radios para hacer esquinas elípticas.

4. - background-image

La propiedad de CSS background-image establece una o más imágenes de fondo en un elemento.

5. - color

La propiedad color de CSS establece el valor de color de primer plano del texto y las decoraciones de texto de un elemento, y establece el valor de color actual. currentcolor puede usarse como un valor indirecto en otras propiedades y es el valor predeterminado para otras propiedades de color, como border-color.

6. - font

La propiedad de CSS font establece todas las diferentes propiedades de la fuente de un elemento. Alternativamente, establece la fuente de un elemento en una fuente del sistema.

7. - font-family

La propiedad CSS font-family especifica una lista priorizada de uno o más nombres de familia de fuentes y/o nombres de familia genéricos para el elemento seleccionado.

8. - font-style, font-weight

La propiedad CSS de font-style establece si una fuente debe tener un estilo normal, cursiva u oblicua de su font-family. La propiedad CSS font-weight establece el peso (o negrita) de la fuente. Los pesos disponibles dependen de la font-family configurada actualmente.

9. - width, height

La propiedad de CSS width establece el ancho de un elemento. De forma predeterminada, establece el ancho del área de contenido, pero si el tamaño del cuadro se establece en border-box, establece el ancho del área del borde.

La propiedad de CSS height establece la altura de un elemento. De forma predeterminada, establece el altura del área de contenido, pero si el tamaño del cuadro se establece en border-box, establece el altura del área del borde.

10. - list-style

La propiedad de CSS de list-style permite establecer todas las propiedades de estilo de lista a la vez.

11. - margin

La propiedad margin de CSS establece el área de margen en los cuatro lados de un elemento.

12. - padding

La propiedad padding de CSS establece el área de relleno en los cuatro lados de un elemento a la vez.

13. -overflow

La propiedad CSS de overflow establece el comportamiento deseado para el desbordamiento de un elemento, es decir, cuando el contenido de un elemento es demasiado grande para caber en su contexto de formato de bloque, en ambas direcciones.

14. -display

La propiedad CSS de display establece si un elemento se trata como un bloque o un elemento en línea y el diseño utilizado para sus elementos secundarios, como el diseño de flujo, la cuadrícula o la flexión.

15. -position

La propiedad position CSS establece cómo se coloca un elemento en un documento. Las propiedades superior, derecha, inferior e izquierda determinan la ubicación final de los elementos posicionados.

16. -filter

La propiedad CSS de filter aplica efectos gráficos como desenfoque o cambio de color a un elemento. Los filtros se usan comúnmente para ajustar la representación de imágenes, fondos y bordes.
